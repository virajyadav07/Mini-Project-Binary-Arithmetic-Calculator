# Mini-Project-Binary-Arithmetic-Calculator

Project Description
This project is a **Binary Arithmetic Calculator** developed using **Scilab**.  
It performs basic arithmetic operations on binary numbers by converting them into decimal, executing the operation, and converting the result back to binary.


Features
- Binary Addition
- Binary Subtraction
- Binary Multiplication
- Binary Division
- User-friendly command-line interface
- Error handling (e.g., division by zero)

 How It Works

1. User inputs two binary numbers.
2. The program converts binary values to decimal using `bin2dec()`.
3. Performs the selected arithmetic operation.
4. Converts the result back to binary using `dec2bin()`.
5. Displays both decimal and binary results.
